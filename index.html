Виртуальный пациент

Веб приложение для изучения клинической фармакологии. Виртуальный пациент подходит для студентов разных факультетов и видов обучения. Приложение содержит в себе тесты и практические задачи для оттачивания навыков и закрепления знаний, полученных в ходе изучения дисциплины.
